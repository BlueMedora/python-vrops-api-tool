### Python scripts to collect data from vrops
 - `brew update`
 - `brew install pyenv`
 - `pyenv init >> .bashrc` (or .zshrc)
 - reload shell
 - `pyenv install 3.5.2`
 - `pip install -r requirements.txt`
 - `./suite-api-tool/tool_ui.py`
